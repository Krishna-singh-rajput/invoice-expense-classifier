# Invoice Expense Classifier API

## 📌 Overview
This project is a Machine Learning based API that classifies invoice text into expense categories.

Categories:
- Logistics
- Office Supplies
- Cloud/Software
- Utilities
- Travel
- Inventory

## ⚙️ Tech Stack

- FastAPI
